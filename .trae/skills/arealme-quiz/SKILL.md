# Arealme 风格角色测试 Skill

这是一个创建类似 arealme.com "测测你是《xxx》中的哪个角色" 风格测试问卷的技能。

## 使用场景

当用户想要创建类似 arealme 风格的测试题，测试用户更像剧集中的哪个角色时使用此 skill。

## 工作流程

### 第一步：获取剧集角色信息

1. 使用 WebSearch 搜索获取角色信息：
   - 搜索 "剧集名称 角色介绍"
   - 搜索 "剧集名称 演员表"
   - 参考 arealme 风格获取角色详细信息

2. 参考以下角色数据格式获取每个角色的：
   - 姓名
   - 头像图片URL
   - 生日、星座
   - 所属组织
   - 能力/技能
   - 经典台词
   - 性格描述

### 第二步：设计二维码类型题目

1. Arealme 使用二维码风格表示角色类型
2. 每个角色分配一个唯一的三位二进制代码（如 111, 101, 011, 000）
3. 代码含义示例：
   - 第一位：性格内向外向
   - 第二位：理性感性
   - 第三位：独立依赖

### 第三步：设计测试题

1. 设计 7-12 道选择题
2. 每道题对应一个维度，每个选项对应 0 或 1
3. 题目类型：
   - 性格偏好题
   - 情境反应题
   - 价值观题

### 第四步：生成 H5 问卷

1. 创建 HTML 文件，数据内联到 JavaScript 中
2. 包含引导页、问卷页、结果页
3. 结果页展示：
   - 角色头像（圆形）
   - 角色名称
   - 二维码类型标识
   - 生日、星座
   - 所属组织
   - 能力描述
   - 经典台词
   - 性格分析

## 输出文件

```
{剧集名称}/
├── data/
│   ├── cast.json          # 演职员表（可选）
│   └── questions.json    # 测试题库
└── quiz.html              # H5问卷页面
```

## questions.json 格式

```json
{
  "drama": "剧集名称",
  "title": "测测你是《剧集名称》中的哪个角色？",
  "description": "简短描述",
  "type": "arealme",
  "characters": {
    "角色名": {
      "name": "角色名",
      "avatar": "头像图片URL",
      "birthday": "生日日期",
      "zodiac": "星座",
      "affiliation": "所属组织",
      "ability": "能力描述",
      "quote": "经典台词",
      "type": "111",
      "description": "性格描述..."
    }
  },
  "questions": [
    {
      "id": 1,
      "text": "问题内容",
      "dimension": 0,
      "options": [
        {"text": "选项1", "value": 0},
        {"text": "选项2", "value": 1}
      ]
    }
  ]
}
```

## H5 实现要点

1. **数据必须内联** - 不要用 fetch 加载外部 JSON
2. **首页展示** - 剧集名称 + 简短描述 + 开始按钮
3. **题目展示** - 进度条 + 当前题目 + 选项
4. **结果页** - 
   - 角色头像（大尺寸，圆形）
   - 角色名称（二维码风格显示 type）
   - 详细信息卡片
   - 重新测试按钮
5. **移动端适配** - 响应式布局
