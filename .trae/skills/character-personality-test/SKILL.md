# 角色人格测试 Skill

这是一个仿造16型人格测试的技能，根据用户指定的剧集创建角色人格测试问卷。

## 使用场景

当用户想要创建类似16型人格的测试题，测试用户更像剧集中的哪个角色时使用此skill。

## 工作流程

### 第一步：获取剧集角色信息

1. 使用 WebSearch 搜索获取角色信息：
   - 搜索 "剧集名称 角色介绍"
   - 搜索 "剧集名称 演员表"
   - 对搜索到的网址进行缓存，然后使用或便捷脚本，获取离角色名称最近的img标签的src属性，作为角色图片URL
   - 获取6-18个主要角色的人设、性格特点

2. 分析每个角色的人格特征：
   - 性格特点
   - 行为模式
   - 价值观
   - 处理问题的方式

### 第二步：创建项目目录

在项目根目录创建：
```
{剧集名称}/
├── data/
└── tmp/   # 可选，用于缓存网页
```

### 第三步：设计测试题

1. 设计20-25道选择题，每题4个选项
2. 每题选项对应不同角色
3. 问题类型：情境题、价值观题、偏好题
4. 不依赖具体剧情细节，保持问题普遍性

### 第四步：生成H5问卷

1. 创建HTML文件，数据内联到JavaScript中
2. 包含引导页、问卷页、结果页
3. 结果页移动端图片宽度≥50%

## 输出文件

```
{剧集名称}/
├── data/
│   ├── cast.json          # 演职员表（可选）
│   └── questions.json     # 测试题库
└── quiz.html              # H5问卷页面
```

## questions.json 格式

```json
{
  "drama": "剧集名称",
  "title": "角色人格测试",
  "description": "测测你更像剧集中的哪个角色",
  "characters": {
    "角色名": {
      "name": "角色名",
      "actor": "演员名",
      "description": "匹配结果描述",
      "traits": ["特质1", "特质2"],
      "quote": "台词",
      "avatar": "🫀",
      "imageUrl": ""  // 可选，有图片则填URL
    }
  },
  "questions": [
    {
      "id": 1,
      "text": "问题内容",
      "options": [
        {"text": "选项1", "character": "角色A"},
        {"text": "选项2", "character": "角色B"},
        {"text": "选项3", "character": "角色C"},
        {"text": "选项4", "character": "角色D"}
      ]
    }
  ]
}
```

## H5实现要点

1. **数据必须内联** - 不要用fetch加载外部JSON
2. **3D球面动画** - 斐波那契球面分布
3. **随机效果** - animationDelay和animationDuration随机值差异要大
4. **移动端适配** - 结果页图片宽度≥50%
