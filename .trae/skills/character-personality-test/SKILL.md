# 角色人格测试 Skill

这是一个仿造16型人格测试的技能，根据用户指定的剧集创建角色人格测试问卷。

## 使用场景

当用户想要创建类似16型人格的测试题，测试用户更像剧集中的哪个角色时使用此skill。

## 工作流程

### 第一步：获取剧集演职员表（含图片）

1. 根据用户输入的剧集名称，首先尝试以下方式获取信息：
   - 使用 WebSearch 搜索 "剧集名称 演职员"获取角色名称、介绍
   - 如果没有合适的，可以尝试"site:movie.douban.com"、"site:tvmao.com"、"m.baike.com"、"maoyan.dianping.com"等等

2. 获取的信息包括：
   - 剧集名称、导演、编剧
   - 所有主要演员及其饰演的角色
   - 角色简介/人设信息

3. **使用爬虫获取角色图片**（两步走）：

   **第一步：缓存网页到tmp目录**
   - 运行 `./python_scripts/scrape.py` 脚本
   - 参数格式：`python scrape.py <剧集名> <URL>`
   - 示例：
     ```bash
     python scrape.py 逐玉 https://www.tvmao.com/kanju/ZGFqI2Fi/actors
     ```
   - 爬虫会自动将网页内容缓存到 `tmp/{剧集名}/` 目录
   - 目前推荐使用电视猫(tvmao)网站，其他网站可能无法正常解析

   **第二步：解析角色图片URL**
   - 运行 `./python_scripts/parse_images.py` 脚本
   - 支持分词输入：用顿号(、)、逗号(，)、空格分隔多个关键词
   - "饰演"作为过滤词，会自动排除包含该关键词的结果
   - 参数格式：`python parse_images.py <剧集名> <关键词>`
   - 示例：
     ```bash
     python parse_images.py 逐玉 樊长玉
     python parse_images.py 逐玉 谢征 言正
     python parse_images.py 逐玉 齐旻
     ```
   - 脚本会在tmp目录的HTML缓存中查找包含关键词的图片
   - 自动将图片URL更新到 `questions.json` 的 `imagePath` 字段
   - 这样H5页面可以通过远程URL直接显示图片

4. 下载保存：
   - 演职员表信息和头像URL（JSON格式保存到 `{剧集名称}/data/` 目录）
   - 网页缓存保存到 `tmp/{剧集名}/` 目录

### 第二步：设计测试题

1. 分析每个角色的人格特征（建议选取6-12个主要角色）：
   - 性格特点
   - 行为模式
   - 价值观
   - 处理问题的方式

2. 设计30题以内的选择题，每题包含4个选项（对应不同角色），确保：
   - 选项涵盖不同角色的典型特征
   - 问题具有普遍性，不依赖剧情细节
   - 问题类型包括：情境题、价值观题、偏好题等

### 第三步：设计评分系统

1. 为每个选项分配角色分值
2. 设计计分逻辑：
   - 用户选择某个选项时，给对应角色加分
   - 统计所有题目后，得分最高的角色即为匹配结果

### 第四步：生成H5问卷

1. 创建HTML文件，包含：
   - 引导页（剧集名称、开始按钮）
   - 每道题目（题目描述、4个选项）
   - 结果页（显示最匹配的角色、头像，分析）
2. 可以使用纯HTML+CSS+JS实现简单的H5页面

## 输出文件

所有生成的文件保存在项目根目录：`{剧集名称}/`

```
{剧集名称}/
├── data/
│   ├── cast.json          # 演职员表数据
│   ├── questions.json     # 测试题库（含角色图片URL）
│   └── actors.json        # 爬虫获取的演员数据
├── images/                # 角色头像目录（可选，本地备用）
│   ├── 角色1.jpg
│   └── 角色2.jpg
└── quiz.html              # H5问卷页面
```

## 爬虫脚本使用说明

### 脚本位置
```
./python_scripts/
├── scrape.py          # 网页缓存脚本
└── parse_images.py   # 图片URL解析脚本
```

### 支持网站（推荐电视猫）
| 网站 | URL示例 | 状态 |
|------|---------|------|
| 电视猫 | `https://www.tvmao.com/kanju/ZGFqI2Fi/actors` | ✅ 推荐 |
| 豆瓣 | `https://movie.douban.com/subject/36554061/celebrities` | ❌ 反爬严重 |

### 使用方法

**第一步：缓存网页**

```bash
cd ./python_scripts
python scrape.py 逐玉 https://www.tvmao.com/kanju/ZGFqI2Fi/actors
```

**第二步：解析角色图片URL**

```bash
# 单个关键词
python parse_images.py 逐玉 樊长玉
python parse_images.py 逐玉 谢征

# 多个关键词（分词搜索）
python parse_images.py 逐玉 谢征 言正
python parse_images.py 逐玉 齐旻

# 为每个角色执行
```

### 分词输入说明
- 支持使用顿号(、)、逗号(，)、空格分隔多个关键词
- 例如：`谢征 言正` 会同时搜索包含"谢征"和"言正"的图片
- "饰演"作为过滤词会自动排除包含该词的结果

### 输出
- 网页缓存：`tmp/{剧集名}/{网站域名}_{路径}.html`
- 图片URL：更新到 `{剧集名}/data/questions.json` 的 `imagePath` 字段

## 角色头像处理

### 方案1：自动解析远程URL（推荐）
- 先运行 `scrape.py` 缓存网页
- 再运行 `parse_images.py` 解析角色图片URL
- H5页面直接使用远程URL显示图片

### 方案2：本地图片备用
- 手动下载图片保存到 `{剧集名称}/images/` 目录
- 文件名格式：`{角色名}.jpg`
- questions.json中imagePath指向本地路径

### 方案3：Emoji后备头像
如果无法获取图片，H5页面会自动显示Emoji头像

## 角色选择建议

- 选择6-12个有鲜明性格特征的角色
- 确保角色之间有明显的性格差异，便于区分
- 优先选择戏份较多，人设完整的角色

## 注意事项

- 确保所有题目的选项与角色特征对应
- 测试结果应该有趣且有参考性
- H5页面需要适配移动端
- 推荐使用电视猫网站进行图片解析
- 如果解析失败，可尝试调整关键词
